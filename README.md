# mapmydomain

mapmydomain is a Python-based tool for subdomain enumeration and sitemap creation.

## Installation

```bash
git clone https://github.com/mohamedicn/mapdomain
```

```bash
pip install -r requirements.txt
```

```bash
python main.py
```
